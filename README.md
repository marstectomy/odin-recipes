# odin-recipes

In this project I will attempt to code a basic HTML website for recipes.

What I hope to learn once I'm done with this project is:

- How to create a repository and clone it to my machine.
- How to code a website using different HTML elements properly.
- How to commit any changes frequently and with good descriptions.
- Get more used to using the Terminal, Github and VScode.

These are my goals!

I finished the project, I guess. While trying to write good commit messages I undid a commit too many times and created a new branch. After much panicking and googling, I managed to find a way to get rid of the new branch and keep my local changes. In the end, I guess I'll have to commit to any commits I do, and it's best to have a bunch of bad commits, than risk messing up the entire project and my project by trying to undo stuff.
